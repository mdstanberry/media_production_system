# channels/

This directory contains publishing specification files for each distribution channel. Each file defines the platform-specific settings, requirements, and publishing checklist for that channel.

One file per channel. The status of each channel is noted at the top of its file.

## Channel Files

| File | Channel | Status |
|------|---------|--------|
| `youtube.md` | YouTube | Active v1.0 |
| `linkedin-video.md` | LinkedIn Video | Planned |
| `podcast.md` | Podcast | Planned |
