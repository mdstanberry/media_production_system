**Status:** planned

## Publishing Specs

| Setting | Value |
|---------|-------|
| Resolution | 1920x1080 (1080p) recommended |
| Frame Rate | 30fps |
| Format | MP4 (H.264) |
| Audio | AAC stereo |
| Max File Size | 5GB |
| Max Duration | 10 minutes |
| Aspect Ratio | 16:9 (landscape) or 1:1 (square) |

## Thumbnail Specs

| Setting | Value |
|---------|-------|
| Size | 1200x627px (16:9) |
| Format | PNG or JPG |

## Naming Convention

- Export: `YYYYMMDD_[topic-slug]_v1.mp4`
- Thumbnail: `YYYYMMDD_[topic-slug]_thumb-A.png`
