# _PROJECT_TEMPLATE/

This folder is the master template for new video projects. **Do not work directly in this folder.**

## How to Start a New Project

1. Copy this entire folder.
2. Rename the copy using the `YYYYMMDD_[topic-slug]/` convention:
   - **YYYYMMDD** — the planned publish date (not today's date)
   - **topic-slug** — lowercase words separated by hyphens

**Example:**
```
20260301_building-data-governance/
```

3. Place the renamed folder inside `projects/`.
4. Begin adding footage and assets to the appropriate sub-folders.

## Folder Contents

| Folder | What goes here |
|--------|---------------|
| `aroll/` | Talking head / camera footage |
| `screen/` | Screen recordings |
| `assets/canva-exports/` | Exported graphics from Canva |
| `assets/reference/` | Reference images, inspiration, brand assets |
| `export/` | Final rendered output files |
| `_COWORK_INSTRUCTIONS.md` | Per-project instructions for the Cowork AI assistant |

## Export File Naming

| Asset | Pattern |
|-------|---------|
| Final video | `YYYYMMDD_[topic-slug]_v1.mp4` |
| Camtasia project | `YYYYMMDD_[topic-slug]_v1.tscproj` |
| Thumbnail A | `YYYYMMDD_[topic-slug]_thumb-A.png` |
| Thumbnail B | `YYYYMMDD_[topic-slug]_thumb-B.png` |
